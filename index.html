Drop one or multiple JSON files at once — each becomes a separate test. Batch upload has no limit.

Best option: upload one JSON file per subject or test. The importer now supports both the old question-array format and a direct subject JSON object with metadata plus questions.
📋 JSON Format
Recommended structure: keep each subject as one JSON object with `title`, `subject`, `description`, `time_per_question`, and `questions`.
{
  "title": "Java OOP - Chapter 4",
  "subject": "Java",
  "description": "Inheritance, abstraction, and interfaces.",
  "time_per_question": 60,
  "questions": [
    {
      "topic": "OOP / Java Basics",
      "question": "Which keyword is used to inherit a class?",
      "options": ["extends", "implements", "inherits", "super"],
      "correct_answer": "extends",
      "explanation": "The extends keyword creates an inheritance relationship."
    }
  ]
}
